# drive_helper
## Goal
A line bot that help rider to determine the safest route.
## Usage
支援三種快速功能 rich menu
1. 路況地圖 產生目前地圖
2. 我的位置 設定目前位置
3. 設定目的地  

開啟地圖後顯示相關資訊  
1. 當前位置 (紅色圖標)
2. 公車動態
3. 推薦路徑
